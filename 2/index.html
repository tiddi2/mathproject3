Truth Table Generator

Info

To generate a truth table we first extracted all the variable names from the equation, then we changed them based on a binary pattern. lets say we have to variables then we iterate from the number 0 to (2^2)-1, and look at the bits in each number to assign the variable to true or false. so 0 = 00 would be false, false, and 1 = 01 would be false, true. then we just add the results to the table

To use this program just write your equation using javascript syntax, so and = &&, or = ||, not = ! and xor = ^. Then you click the generate button, just make sure to not use too many variables at around 12 variables it will start to really slow down
